# Week 2 - Session 2

## Podman vs Docker
- Podman: rootless, secure, Docker-compatible
- Docker: mature ecosystem, requires root daemon

## GitHub Pages
- Host static sites from repos
- Push to `main` or `gh-pages` branch
- Great for docs, portfolios, projects

## FastAPI
- Async Python framework
- Auto docs (Swagger/ReDoc)
- Ideal for microservices & prototyping

## Vercel
- Zero-config cloud deployment
- CI/CD ready
- Supports Next.js, static sites, APIs
